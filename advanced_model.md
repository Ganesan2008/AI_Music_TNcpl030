# Emoji Music Composer

## Overview
Emoji Music Composer is an AI-powered tool that generates music based on emojis or text inputs. Leveraging state-of-the-art natural language processing and deep learning techniques, this tool transforms simple inputs into unique musical compositions.

## Features
- **Music Generation**: Create music by entering emojis or text.
- **Advanced Modeling**: Utilizes the Facebook MusicGen model for sophisticated music generation.
- **Emoji Support**: Supports a wide array of emojis with descriptive mappings to facilitate diverse musical creativity.
